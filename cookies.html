How these controls work

This page stores a consent record in your browser using local storage. The supplied JavaScript exposes your selections through window.foxTimberConsent and dispatches a foxTimberConsentChanged event.

Optional analytics, advertising, embeds, or other trackers must be configured to check that consent record before loading. Simply adding this page does not automatically control scripts added by third-party services.

You can return to this page at any time to change or withdraw optional consent. Clearing browser storage will reset the choices on that browser.
